
# RaspberryPi Project
## Psyren

### ✈️ 소개
본 프로젝트는 교내 캡스톤 디자인 프로젝트로 라즈베리파이를 이용한 청각장애인용 스마트 디스플레이를 제작하여
청각장애인이 일상생활에서 겪는 불편함 해소를 위해 소리 분류, 음성인식, 수어번역 서비스를 제공하고자 한다.
<br/>

### 📌 기술
- Python
- speech_recognition/gTTS
- MedieaPipe
- KNN
- openCV

### 📌 주요 기능
- 티쳐블 머신을 사용한 주변 위험 소리 인식 기능
- STT/TTS를 이용한 양방향 소통 기능
- 수어 번역을 이용한 소통 기능
